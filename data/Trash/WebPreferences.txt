---++ Appearance

   * Set WEBBGCOLOR = #FF0099
   * Set WEBSUMMARY = Place for trashed topics and attachments.
   * Set NOSEARCHALL = on

---++ Access control

Restricted to adminsso that deleted topics can't be viewed by spies.

   * Set ALLOWWEBVIEW = %USERSWEB%.AdminGroup
   * Set ALLOWTOPICCHANGE = %USERSWEB%.AdminGroup
   * Set ALLOWTOPICRENAME = %USERSWEB%.AdminGroup

----++ Finalisation

   * Web preferences that are *not* allowed to be overridden by sub-web, user or topic preferences:
      * Set FINALPREFERENCES = NOSEARCHALL, ATTACHFILESIZELIMIT, WIKIWEBMASTER, WEBCOPYRIGHT, DENYWEBVIEW, ALLOWWEBVIEW, DENYWEBCHANGE, ALLOWWEBCHANGE, DENYWEBRENAME, ALLOWWEBRENAME
